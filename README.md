```bash

```

```bash
  yarn install my-project
  cd my-project
```

<!-- Running Tests -->

### :test_tube: Running Tests

To run tests, run the following command

```bash

```

<!-- Run Locally -->

### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/harun181/c_programming
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
